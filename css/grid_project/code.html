CSS Grid in Code

How it works...

  1. To use this project you will need a CSS compiler, a text editor and version control software.
  2. Use the following commands in your CLI.
  3. Clone the repository to a directory on your system. git clone ...
  4. Use sass --watch style.scss:style.css to operate sass and make changes to SASS Code.
  5. Open a sass partial text file. e.g. grid_areas.scss
  6. Swap out the import file and review the changes made to index.html
  7. Use code.html for more information about what is happening with each sass partial.

@import "template_columns.scss" Grid Template Columns

This generates rows of 3 items (for as long as there are grid items to fill them).

Where there is no space on a line it rolls over onto the next lime. Where there is no element to go into a grid space it is simply left clear with no need to floats or float clearing.

No need for floats is one of the benefits of using CSS Grid with no need for disruption to normal document flow.

@import "grid_order.scss" Grid order

For this to work, I have set up a sass partial file that overrides the selectors so that the grid areas are removed, assigned template columns and used the order property with a number of 1-6 on the grid items.

The inherit property that is included in the grid items is important to allowing the changes to take effect.

@import "grid_row.scss" Grid Row

Grid Row property sets the size of a grid item and its location within a grid row. They are designed to specify on which row to start displaying the item and on how many rows to span the contents of the grid.

There are 2 properties for grid grow. They can be short handed to 2 integer values.

In this example, the shorthand property of 1/4 seems to span the 3 rows that the rest of the grid items make up. Each other grid item has been given a grid-are of inherit to account for the default grid areas.

@import "grid_implicit.scss"Explicit and Implicit Grid

Using Grid properties like template-rows and template-columns creates a defined or explicit grid.

If a grid item is used that is outside of the range of the grid-template-rows or columns it becomes an implicit grid that is generated by the browser.

@import "grid_repeat.scss"Repeat Notation

Using the repeat() function allows template columns or rows to be more quickly and succinctly. It is useful for defining repeating and repetitive grid line tracks

Define the number of times you want an expression to repeat and then define the expression.

import "justify-self.scss"Grid Box Alignment - justify-self

The the alignment an placement of grid items. In this example, the item with center justification is centered in its container and no longer stretches to take up all of the available space.

import "minmax.scss"minmax()

The minmax() is a useful function that allows you define a minimum and maximum space that a grid item takes up inside its grid container. the second is the maximum. Using 1fr as the maximum value the space is equally distributed and takes up as much available space as possible relative to their sibling elements.

My example shifts the middle column in size so it takes up a minimum and a maximum, in between 1fr on the left and the right hand side.
