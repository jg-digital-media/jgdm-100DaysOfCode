@import "grid_row.scss" Grid Row

Grid Row property sets the size of a grid item and its location in the grid. They are designed to specify on which row to start displaying the item and on how many rows to top or span the content of the grid.

There are 2 properties for grid grow. They can be short handed to 2 integer values.

A shorthand property of 1/4 seems to span the 3 rows that the rest of the grid items make up. Each other grid item has been given a grid-are of inherit to account for the default grid areas.
