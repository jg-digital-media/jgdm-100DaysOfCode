Class List -

In Simple Terms, the classList method is a read only method in the DOM that returns a list of names of classes associated to a specified HTML element.

It is useful to add and remove CSS classes and by extension the visible and invisible states of elements for responsive navigation.

The classList property is read-only, however, you can modify it by using the add() and remove() methods.

  • Add a class - el.classList.add("open");
  • Add Many Classes - el.classList.add("this", "little", "piggy");
  • Remove a Class - el.classList.remove("open");
  • Remove Multiple Classes - el.classList.remove("this", "little", "piggy");
  • Integer of the number of classes in an element - el.classList.length
  • el.classList.replace("is-big", "is-small");
  • Replace a class (replaces first with second class string) - el.classList.replace("is-big", "is-small");
  • Returns true of false - el.classList.contains("open");
  • Toggle a class - remove if there... add if not - el.classList.toggle("open");
