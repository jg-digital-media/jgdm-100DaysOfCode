Matching Specific Characters

Regex - lady ?bugs?

  • ladybug
  • ladybugs
  • lady bugs

Match literal characters

? matches 0 or more of the previous characters

Regex - la[a-z]y ?[a-z]ugs?

  • ladybug
  • lady bugs
  • lazy bug
  • lazy lug

Match literal characters

[] character ranges - match the characters or range of characters in the set

Regex - [lh]a[a-z]y[ ]?[a-z]?l?ugs?

  • ladybug
  • lazy lug
  • lazy slug
  • hazy slug

Regex - [fladying]+ ?[br]ug!?

  • ladybug
  • fading rug!

+ - match preceding character from once to as many times as possible: unlimited.

  • ladybug
  • lazy lug
  • lazy slug
  • hazy slug

Matching Character Ranges

Regex - [a-z]+

  • bat
  • cat
  • data
  • eat
  • fast

Character ranges within character sets.

Matches 5 times, one for each word with +.

Regex - [0-9] ?[a-b]

  • 1 a
  • 2 a
  • 3 a
  • 4 a
  • 5 a
  • 8 a
  • 9 b

Character ranges within character sets.

Matches 7 times, one for each word. Optional (0 or 1) space with ?

Regex - [0-9]+ ?[a-zA-z]+
