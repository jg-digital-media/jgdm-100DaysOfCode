Introduction

This website uses feature detection with Modernizr.js

Integrating Modernizr into a website is very simple. You add what you want to detect on the Modernizr website and download the generated file. It's process that when applied, enhances the site's ability to handle feature detection, ensuring a smooth experience across different browsers.

Using Modernizr's JavaScript API

To make better use of Modernizr, you should:

  1. Identify which features you need to detect (CSS3, HTML5 features, etc.)
  2. Create a custom build of Modernizr that only includes the tests you need
  3. Use Modernizr's classes in your CSS for graceful degradation
  4. Use Modernizr's JavaScript API to provide fallbacks when needed
