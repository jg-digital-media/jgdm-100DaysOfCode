# SEO Checklist

## Basic Technical Setup

Add and verify the site in Google Search Console

Add the site to Bing Webmaster Tools

Submit the XML sitemap

Ensure the robots.txt file is present and correct

Check the site is indexable (no noindex left from staging)

Confirm canonical tags are correct

Set the preferred domain (www or non-www)

Force HTTPS and check redirects

Check 301 redirects from any old URLs

## Page-Level SEO

Write a unique title tag for every page

Add meta descriptions to key pages

Use one H1 per page

Check heading structure (H1 → H2 → H3)

Ensure URLs are clean and readable

Add internal links between important pages

Check for duplicate content

## Content & Keywords

Confirm each page targets a primary keyword/topic

Ensure keywords appear naturally in titles and headings

Add alt text to images

Check content is original and helpful

Make sure key pages have sufficient content

## Performance & Technical

Test page speed

Optimise images

Enable compression (Gzip/Brotli)

Check mobile friendliness

Fix any broken links

Check for crawl errors

## Structured Data & Social

Add structured data (Schema markup) where relevant

Add Open Graph tags for social sharing

Add Twitter/X card tags

Test how pages look when shared on social media

## Analytics & Tracking

Install Google Analytics

Confirm analytics tracking is firing

Set up conversion goals or events

Link Search Console with Analytics

## Final Pre-Launch Checks

Check favicon appears

Create 404 page

Ensure contact information is visible

Add privacy policy

Add cookie notice if required

Test forms and CTAs
